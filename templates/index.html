Datacentralen

Denna webbsida demonstrerar hur man kan hämta och visa data från ett API med tre olika tekniker: Python (via requests), JavaScript fetch() och axios. Syftet är att jämföra hur dessa metoder fungerar i praktiken - både i hur de implementeras och i hur snabbt de levererar data. Prestandamätning kan du se via DevTools (förutom i requests) där den hur lång tid det faktiskt tar att hämta informationen. I det här fallet är det städer som kommer vara själva datan.

  • Python (requests)

    requests är ett Python-bibliotek för att skicka HTTP-förfrågningar. Det används oftast på serversidan för att hämta data från API:er

  • JavaScript (fetch)

    fetch är en inbyggd webbläsar-API som används för att hämta data från servrar i JavaScript.

  • JavaScript (axios)

    axios är ett externt JavaScript-bibliotek som gör det ännu enklare och kraftfullare att göra HTTP-anrop än med fetch.
